# SocialOps API

Dedicated API surface for the local AI video/content engine.

Initial responsibilities:

- auth and workspace source-of-truth
- profiles and projects
- capture notes
- content drafts
- content-engine purpose and multi-platform plan
- x-algorithm draft ranking
- approvals
- metrics and learning loop
- job orchestration across MiniClaw, ComfyUI, ComfyUI extensions,
  PokeeResearch, x-algorithm, Postiz/OpenPost, deck, and video workers

Posting/publishing should use connected services and official platform-supported
flows through Postiz/OpenPost or manual workflows.
